About me

Teknis, praktis, dan suka bikin sistem yang cepat jadi berguna.

Saya terbiasa berada di antara network operation, monitoring, GPON provisioning, dan pembangunan tools internal. Fokus saya bukan hanya menyelesaikan problem teknis, tapi juga memastikan hasilnya enak dipakai dan relevan buat tim lapangan.

Siapa saya

Saya adalah network engineer / NOC specialist dengan ketertarikan kuat pada routing, reliability, observability, GPON/OLT operation, dan automasi. Saya nyaman mengerjakan persoalan teknis yang butuh pemahaman lintas area: jaringan, sistem, dashboard, dan workflow operasional.

Gaya kerja

Saya lebih suka eksekusi cepat dan iteratif. Buat saya, solusi terbaik biasanya dimulai dari versi minimum yang langsung usable, lalu disempurnakan sambil melihat kebutuhan riil dari pengguna.

Hal yang sering saya kerjakan

  • • Routing, BGP, VLAN, QoS, firewall, dan troubleshooting jaringan
  • • Monitoring jaringan dan sistem, alerting, dashboard, webhook
  • • Provisioning OLT/ONT GPON, registrasi SN/LOID, profiling layanan
  • • Tool internal untuk billing, status operasional, klinik, dan automasi workflow

Yang saya suka dari project teknis

Saya suka saat sebuah project benar-benar mengurangi kerja manual, mempercepat troubleshooting, atau bikin tim lebih mudah memahami kondisi sistem. Buat saya itu jauh lebih memuaskan daripada sekadar demo yang terlihat keren.

⚙️

Ops-first mindset

Keputusan teknis saya biasanya ditimbang dari sisi operasional: apakah lebih mudah dipantau, lebih cepat ditindak, dan lebih ringan dijalankan tim.

🔁

Iterasi cepat

Saya suka membuat versi awal yang usable dulu, lalu memperhalus UX, data, monitoring, dan automasi setelah dipakai secara nyata.

🧩

Nyaman lintas domain

Mulai dari routing dan monitoring sampai dashboard internal dan API integration, saya nyaman bergerak di area yang saling bertemu ini.
